# Sample CLI Rust

> gitブランチ名作成コマンドラインツール

## Usage

```bash
$ sample-cli-rust [FLAGS]

FLAGS:
    -d, --debug      デバッグモード
    -e, --execute    gitコマンド実行
    -h, --help       display help information
    -v, --version    display version information
```

## Development

```bash
$ git clone https://github.com/nakamura0907/sample-cli-rust
$ cd sample-cli-rust
$ cargo build
```
